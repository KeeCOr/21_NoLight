# 2026-05-18 Shark Scale Combat Design

## 목표

플레이 화면에서 비어 보이는 영역을 줄이기 위해 기본 실행 해상도를 `1080 x 1920`으로 확대한다. 전투력은 생명력 100 기준에서 직관적으로 읽히는 20~40대 값으로 조정하고, 정면 충돌에서는 전투력 차이만큼 약한 쪽의 생명력이 감소하도록 만든다.

## 규칙

- 플레이어 기본 전투력은 24다.
- 일반 상어 적은 18~27 전투력을 가진다.
- 추격 상어는 42 전투력을 가진다.
- 정면 힘싸움은 두 전투력의 차이를 피해량으로 사용한다.
- 후방 공격은 약한 상어도 이길 수 있도록 공격자 전투력의 1.5배 피해를 준다.
- 물고기 보상 하나를 먹을 때마다 전투력이 3 오른다.
- HUD에는 현재 플레이어 전투력을 `PWR`로 표시한다.
- 적 머리 위에는 `전투력/현재 생명력` 형식의 숫자를 표시한다.

## 구현 단위

- `src/systems/SharkCombat.js`: 전투력 조회, 후방 판정, 정면 힘싸움, 물고기 성장 계산을 담당한다.
- `src/entities/ElectricCharacter.js`, `src/entities/MechaArmCharacter.js`: 기본 공격에 상어 전투 규칙을 적용한다.
- `src/entities/Enemy.js`, `src/entities/Pursuer.js`: 접촉 전투를 전투력 차이 기반으로 처리한다.
- `src/scenes/GameScene.js`: 물고기 보상 획득 시 HP 회복과 전투력 성장을 같이 처리한다.
- `src/ui/HUD.js`: 플레이어 전투력을 화면에 표시한다.
- `src/game.js`, `main.js`: 기본 화면 크기를 `1080 x 1920`으로 확대한다.

## 테스트

- 정면 힘싸움 피해량이 전투력 차이와 일치하는지 검증한다.
- 후방 공격 시 약한 상어가 강한 상어에게 유효 피해를 주는지 검증한다.
- 물고기 성장으로 전투력이 작은 단위로 증가하는지 검증한다.
- Electron 창 크기가 Phaser 캔버스 크기와 일치하는지 검증한다.
