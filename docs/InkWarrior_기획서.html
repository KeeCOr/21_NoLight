InkWarrior 기획서 v0.3

버전: 0.3.0 | 최종 수정: 2026-06-24 | 출시 플랫폼: Windows PC portable, Web/Electron 로컬 빌드

문제 정의

짧은 액션 세션에서 플레이어는 공격, 회피, 피해, 경직, 처치 결과를 즉시 읽고 싶다.

게임 소개

InkWarrior는 수묵화 질감의 횡스크롤 액션 플랫포머다. 플레이어는 캐릭터를 교체하며 무한 청크 맵을 올라가고, 적과 추격자를 상대해 생존 시간과 점수를 늘린다.

핵심 루프

상황 확인 → 행동 선택 → bitmap VFX와 콜아웃 피드백 → 보상/손실 확인 → 다음 전투

게임 플레이 예시

  1. 공격 키를 누르면 붓길 예고와 brush slash VFX가 나온다.
  2. 명중하면 참격 콜아웃, 먹 폭발, 카메라 흔들림, hit-stop이 발생한다.
  3. 피해를 입으면 먹번짐 경직과 blood ink VFX가 표시된다.
  4. 적을 처치하면 먹물 폭쇄와 impact ink burst가 나오고 회복 아이템이 생성된다.

주요 시스템

  • 캐릭터 교체
  • 수묵 전투 피드백

  • 무한 청크 맵
  • 추격자 압박
  • 회복 드롭

개선 제안

아래 제안은 현재 기획을 바탕으로 도출한 우선순위별 개선 방향입니다.

🎮 게임플레이 / 핵심 기능

우선순위 제안 기대 효과
높음 첫 플레이 시 적과 마주치는 순간 공격·회피·교체 키 힌트를 화면 하단에 반투명 오버레이로 표시하고, 첫 명중 콜아웃 후 자동 제거 수묵 전투 언어(strike/evade/wound/finish)를 사전 설명 없이 체감으로 학습하게 해 이탈 감소
높음 추격자와의 거리를 화면 우측 에지에 색상 바(녹→황→적)로 실시간 표시하고, 위험 거리 진입 시 카메라 가장자리를 붉은 먹 번짐으로 비네팅 처리 추격자 압박이 시각·직관적으로 전달되어 탈출/교전 판단 속도 향상
중간 청크 전환 시 10% 확률로 "도망칠 수 없다" 특수 밀폐 방(출구가 잠기고 적 웨이브 격파 후 개방)을 삽입해 강제 교전 긴장감 생성 단순 올라가기 반복 피로를 줄이고 세션 중 고조점을 만들어 점수 경쟁 동기 부여
중간 처치 연속 횟수(콤보)를 청크 단위로 집계해 콤보 임계치(5·10·20연)마다 붓 흔적 개수가 화면을 수직으로 가르는 "참격 컷씬" 연출 삽입 콤보 유지 플레이를 명확한 보상으로 강화해 기술 숙련 동기 부여
낮음 전기형·메카 암 외 세 번째 캐릭터를 "먹구름 회피형"으로 추가하고, 삼각 교체 상성(전기→약→메카, 메카→약→먹구름, 먹구름→약→전기)을 적 종류에 부여 2캐릭터 이분법에서 3캐릭터 가위바위보 구조로 전략 깊이 확장

🖥️ UX / UI

  • 콜아웃 과부하: 같은 프레임에 3개 이상 콜아웃이 겹칠 경우 가장 높은 우선순위 1개만 표시하고 나머지는 소형 아이콘으로 우측 스택 — 수묵 화면 가독성을 유지하면서 주요 피드백 누락 없이 전달
  • HP·스태미나 바 가독성: 현재 수묵 배경에 얇은 바가 묻히는 문제를 방지하기 위해 붓질 텍스처 테두리 + 흰 번짐 후광으로 대체하고, 스태미나 소진 시 바 테두리를 1초간 진동 처리
  • 처치/회복 아이템 연결성: 아이템 생성 위치에서 플레이어 방향으로 먹 물줄기 유도선을 0.3초간 표시해 "처치→회복 드롭→획득" 인과를 시각화, 놓치는 아이템 수 감소

💰 수익화 전략

  • 세션 후 스킨 해금: 생존 시간·최고 콤보 기준으로 플레이 후 수묵화 스타일의 칼 스킨, 잉크 색상 팔레트(세피아, 청묵, 홍묵) 해금 — 반복 플레이 동기를 외부 결제가 아닌 숙련 보상 구조로 설계해 무료 유저 유지율 선확보 후 유료 DLC 확장
  • 유료 캐릭터 DLC: 세 번째 캐릭터 이후부터 "필체 유파" 테마 캐릭터(해서체형, 초서체형 등)를 1~2달러 단건 DLC로 판매 — 수묵 테마에 부합하는 네러티브가 있어 이질감 없이 컬렉션 욕구 자극
  • 리더보드 시즌 패스: 월간 시즌 리더보드에 상위 랭커 전용 수묵 프레임 보상을 추가하고, 시즌 패스(3달러) 구매자에게 중간 보상 2단계 추가 제공 — 경쟁 구조로 DAU 유지

📦 콘텐츠 확장

  • 단기(1~2개월): BGM/SFX 오디오 시스템 통합(참격·피격 음이 수묵 VFX와 프레임 동기), 특수 이벤트 청크 3종(낙하 구간, 가시 벽 이동, 보스 전위 전투) 추가
  • 중기(3~6개월): 세 번째 캐릭터(먹구름 회피형) 추가 및 3방향 상성 시스템 구현, 일일 도전 모드(지정 캐릭터·제한 조건으로 특정 점수 달성), 수묵 팔레트 스킨 10종 해금 트리
  • 장기(6개월+): 협동/경쟁 멀티플레이(같은 맵을 두 플레이어가 다른 경로로 동시 진행 후 점수 합산), 커뮤니티 제작 청크 에디터 공개, 전통 회화 작가 협업 한정 아트 팩

⚙️ 기술적 개선

  • 청크 생성 시드 저장: 현재 무한 청크가 실행마다 랜덤 재생성되는 구조를 시드 기반으로 전환해 특정 명국을 리플레이하거나 공유할 수 있게 함 — 소셜 바이럴 경로 확보
  • ActionFeedback 우선순위 큐: 같은 프레임에 여러 ActionFeedback 이벤트가 발생할 때 처리 순서를 명시적 우선순위 큐로 관리해 콜아웃·VFX 덮어쓰기 버그 방지
  • CameraImpactProfile 런타임 튜닝 UI: 개발 빌드에서 흔들림/줌/플래시 파라미터를 슬라이더로 실시간 조정할 수 있는 디버그 패널을 추가해 QA 시간 단축 및 느낌 조정 반복 비용 절감
