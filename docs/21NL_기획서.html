InkWarrior 기획서 v0.3

버전: 0.3.0 | 최종 수정: 2026-06-24 | 출시 플랫폼: Windows PC portable, Web/Electron 로컬 빌드

문제 정의

짧은 액션 세션에서 플레이어는 공격, 회피, 피해, 경직, 처치 결과를 즉시 읽고 싶다.

게임 소개

InkWarrior는 수묵화 질감의 횡스크롤 액션 플랫포머다. 플레이어는 캐릭터를 교체하며 무한 청크 맵을 올라가고, 적과 추격자를 상대해 생존 시간과 점수를 늘린다.

핵심 루프

상황 확인 → 행동 선택 → bitmap VFX와 콜아웃 피드백 → 보상/손실 확인 → 다음 전투

게임 플레이 예시

  1. 공격 키를 누르면 붓길 예고와 brush slash VFX가 나온다.
  2. 명중하면 참격 콜아웃, 먹 폭발, 카메라 흔들림, hit-stop이 발생한다.
  3. 피해를 입으면 먹번짐 경직과 blood ink VFX가 표시된다.
  4. 적을 처치하면 먹물 폭쇄와 impact ink burst가 나오고 회복 아이템이 생성된다.

주요 시스템

  • 캐릭터 교체
  • 수묵 전투 피드백
  • CameraImpactProfile 순수 규칙
  • 무한 청크 맵
  • 추격자 압박
  • 회복 드롭

현재 구현 상태

  • 현재 문서 기준 버전: 0.3.0
  • 전체 게임 루프, 캐릭터 교체, 무한 청크 맵, 적/추격자 전투 구현 완료
  • 공격/회피/피격/경직/패배 피드백 규칙을 ActionFeedback으로 통합
  • CameraImpactProfile 순수 규칙으로 카메라 타격 연출 관리
  • UI 디자이너 후속 작업 목록: docs/ui-designer-camera-impact-tasks.md

남은 리스크와 다음 우선순위

  2. 세 번째 캐릭터를 추가한다.

  4. 특수 지형/이벤트 청크를 추가한다.
  5. 전투 피드백 규칙을 실제 플레이 녹화로 검증한다.
