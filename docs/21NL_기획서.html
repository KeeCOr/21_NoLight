InkWarrior 기획서

현재 문서 기준 버전: 0.4.0

문제 정의

짧은 세션에서 빠르게 손맛을 느끼고 싶은 PC 액션 플레이어가 공격, 회피, 피격, 처치 결과를 즉시 읽지 못하면 전투가 밋밋하게 느껴진다. InkWarrior는 수묵 VFX, 카메라 반응, 적 리액션을 같은 프레임에 묶어 전투 확신을 주는 액션 플랫폼 게임이다.

현재 구현 상태

  • 현재 문서 기준 버전: 0.4.0
  • CameraImpactProfile 순수 규칙으로 카메라 흔들림, 줌, 플래시, 방향 넛지를 관리한다.
  • ComboHitReaction 순수 규칙으로 1타/2타/3타 적 넉백, 경직, smear, 피니셔 플래시, 드랍 지연을 분리했다.
  • UI 디자이너 후속 작업 목록은 docs/ui-designer-camera-impact-tasks.md에 정리했다.

핵심 루프

추격자와 적 배치를 확인한다 → 공격, 회피, 스킬, 캐릭터 교체를 선택한다 → 수묵 VFX와 적 리액션으로 결과를 읽는다 → 회복 드랍과 점수를 챙긴다 → 다음 청크로 이동한다.
