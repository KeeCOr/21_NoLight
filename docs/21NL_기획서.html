  • 기본 실행 해상도를 1080 x 1920으로 확대해 화면의 비어 보이는 영역을 줄였다.
  • 전투력은 생명력 100 기준에서 읽기 쉬운 20~40대 숫자로 재조정했다.

  • HUD에는 현재 플레이어 전투력 PWR을 표시하고, 적 머리 위에는 전투력/현재 생명력을 표시한다.

묵철 (墨鐵)

수묵화 스타일 2D 액션 플랫포머

프로젝트: 21_NL  |  v0.1.3 Windows (Electron + Phaser.js)  |  최종 수정: 2026-05-29

1. 게임 개요

동양 수묵화 배경에 메카닉 요소가 혼합된 캐릭터가 등장하는 2D 액션 플랫포머.

플레이어는 2명의 캐릭터를 실시간으로 교체하며 전투하고, 강력한 추격자를 피해 무한 생성 맵을 이동한다.

명확한 엔딩 없이 최대한 오래 생존하며 점수를 쌓는 서바이벌 방식이다.

점수 = 생존 시간(초) x 10 + 처치 수 x 50

키동작
방향키 좌/우이동 (속도 220)
방향키 위점프 (지면 접촉 시만, 속도 -550)

Space게임 시작 / 재시작
ESC메인 메뉴 복귀 (게임오버 시)

3. 핵심 메커니즘

3-1. 저체력 강화 (Risk & Reward)

적을 처치하지 않으면 HP가 지속 감소. HP가 낮을수록 강해진다.

조건효과
HP 30 이하 (저체력)공격력 1.5배
HP 30 이하 (저체력)스태미나 자동 충전 2배 (초당 10 -> 20)
적 미처치 3초 이상 경과HP 초당 2씩 감소
적 처치 시HP 감소 3초간 유예

3-2. 캐릭터 교체

  • Tab 키로 즉시 교체
  • 교체 순간 새 캐릭터의 onSwapIn() 특수 효과 발동
  • HP와 스태미나는 두 캐릭터가 공유
  • 교체된 캐릭터는 이전 캐릭터의 위치에 등장
  • 캐릭터 풀 구조로 확장 용이

3-3. 스태미나

상황수치
최대치100
자동 충전 (일반)초당 10
자동 충전 (저체력)초당 20
가드 유지 중 소모초당 15
스킬 발동 소모30
메카 팔 가드 피격 시 획득20

3-4. 패링

  • 가드(C키) 입력 후 100ms 이내 피격 시 패링 판정
  • 패링 성공: 투사체 반사 + 적 공격 캔슬
  • 두 캐릭터 공통 적용

3-5. 무적

  • 피격 시 500ms 무적 (반투명 표시)
  • 캐릭터별 특수 무적 시간 별도 존재

4. 캐릭터 시스템

전기 범위형 (ElectricCharacter)

범위 공격과 역장으로 다수의 적을 제어

행동설명수치
공격 (Z)전방 전기 방출 + 스턴20dmg, 200범위, 500ms스턴
스킬 (X)광역 전기 폭발 + 스턴50dmg, 300범위, 1000ms스턴
가드 (C)역장: 투사체 30%로 감속250범위, 대미지 50% 경감
교체 진입주변 적 감전 스턴200범위, 800ms스턴

메카 팔 근접형 (MechaArmCharacter)

빠른 콤보와 대시로 근접전 특화

행동설명수치
공격 (Z)3단 콤보 (600ms내)15dmg, x1.0/1.2/1.5, 80~120범위
스킬 (X)대시 돌진 + 무적속도800, 200ms무적
가드 (C)풀 대미지 + 스태미나 충전100% 대미지, +20 스태미나
교체 진입무적 + 속도 부스트800ms무적, 1.5배속(1초)

5-2. 추격자 (Pursuer)

게임 내내 플레이어를 추적하는 파괴 불가 보스급 존재.

항목수치
기본 이동 속도120
접촉 대미지5 (0.5초마다)
접촉 판정 거리60
중력없음 (공중 이동)

공격 패턴 (순환)

순서패턴간격대미지
1투사체 발사 (플레이어 조준)3초15 (속도 300)
2범위 충격파 (예고 0.8초 후)6초20 (반경 200)
3돌진 (플레이어 방향)8초접촉 대미지 (속도 500, 0.4초)

저지 메커니즘

  • 누적 대미지가 임계값 도달 시 4초간 정지
  • 초기 임계값: 200, 저지 성공마다 +50 증가
  • 패링 투사체 반사 시 추격자에 30 대미지

난이도 곡선

  • 30초마다 이동 속도 5% 증가

6. 맵 시스템

청크 기반 무한 생성

  • 맵은 청크(Chunk) 단위로 관리 (높이 300px)
  • 플레이어 위치 기준 상하 2개 청크 범위 유지
  • 범위 밖 청크는 자동 제거
  • 최초 청크(인덱스 0)에는 적 미스폰

발판 배치

항목수치

12. UX 온보딩 시스템 (v0.1.0 추가)

3스텝 조작법 오버레이

게임 시작 직후 자동으로 표시되는 3단계 조작법 안내 오버레이. 최초 1회만 노출 (onboardingSeen = true 저장 후 재노출 없음).

스텝안내 내용
1이동 (방향키 좌/우), 점프 (위 방향키)
2공격 Z / 스킬 X / 가드 C(홀드)
3Tab — 캐릭터 교체 (즉시 효과 발동)

캐릭터 스왑 첫 노출 힌트

첫 번째 캐릭터 교체(Tab) 전까지 HUD 우측 하단 캐릭터 아이콘 위에 깜빡이는 힌트 라벨 "Tab으로 교체" 표시. 교체 1회 성공 시 사라짐.

연습 스테이지 가이드

구간 0 (첫 청크)에서 적이 스폰되지 않는 동안 화면 중앙 상단에 단계별 행동 유도 텍스트를 순서대로 노출:

  1. "← → 로 이동해 보세요"
  2. "Z 키로 공격해 보세요" (다음 청크 첫 적 등장 전)
  3. "Tab 으로 캐릭터를 바꿔 보세요" (첫 적 처치 후)
